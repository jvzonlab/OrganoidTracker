organoid_tracker.core.link_data module

class organoid_tracker.core.link_data.LinkData(forward: Links)

Bases: object

All link metadata used to be stored in this class. Now it is stored directly in Links, so you can access it using experiment.links (instead of experiment.link_data). This class only exists for backwards compatibility, and acts as a thin wrapper of the Links class.

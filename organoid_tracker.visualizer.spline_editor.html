organoid_tracker.visualizer.spline_editor module

class organoid_tracker.visualizer.spline_editor.SplineEditor(window: Window)

Bases: AbstractEditor

Editor for splines. Double-click to (de)select a spline. Draw the spline by adding points using the Insert key. Use Delete to delete a spline. Press C to copy a selected spline from another time point to this time point. Press P to view the position of each cell on the nearest spline.

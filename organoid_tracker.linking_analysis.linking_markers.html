Returns True if the given position is marked as uncertain. A person can mark a position as uncertain if it is not clear whether there is actually a position marker there. The error checker will warm for such cases.

organoid_tracker.linking_analysis.linking_markers.remove_error_marker(positions: PositionCollection, position: Position, error: Error)

Removes an error marker if (and only if) it’s currently set to the given error.

See also:

  • set_error_marker(positions, position, None) - removes any error marker, independent of the current one

  • suppress_error_marker(positions, position, error) - makes this position immune of the given type of error, even if set_error_marker is called, it will not show up in get_error_marker. Normally used by the end-user to suppress false positives.

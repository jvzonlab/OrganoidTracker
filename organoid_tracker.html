Module contents

This is the package of OrganoidTracker, where all the code is stored.

To load an experiment, use:

>>>    from organoid_tracker.imaging import io
>>>    experiment = io.load_data_file("some_file_name.aut")

Head over to the organoid_tracker.core.experiment.Experiment class to see how the experimental data is structured.

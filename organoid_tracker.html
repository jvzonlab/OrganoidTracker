Module contents

OrganoidTracker is a program to automatically (or manually) track cell nuclei.

To load an experiment, use:

from organoid_tracker.imaging import io experiment = io.load_data_file(file_name)

Head over to the core.experiment.Experiment class to see how the experimental data is structured.

class organoid_tracker.core.shape.EllipseShape(ellipse_dx: float, ellipse_dy: float, ellipse_width: float, ellipse_height: float, ellipse_angle: float, original_perimeter: float = - 1, original_area: float = - 1, eccentric: bool = False)

Bases: ParticleShape

Represents an ellipsoidal shape. The shape only stores 2D information. This class was previously used to store some primitive shape information. The class is still present, so that you are able to load old data.

draw2d(x: float, y: float, dz: int, dt: int, area: Axes, color: Union[Tuple[float, float, float], Tuple[float, float, float, float], str, float], edge_color: Union[Tuple[float, float, float], Tuple[float, float, float, float], str, float])

Draws a shape in 2d, at the given x and y. dz is how many z layers we are removed from the actual position, dt the same, but for time steps. Both dz and dt can be negative. area is the drawing surface, and color is the desired color.

draw2d_image(x: float, y: float, dz: int, dt: int, image: ndarray, color: Tuple[float, float, float])

Draws the shape to a 2d array. The shape of the array is (y_size, x_size, 3), with the last channel being the color.

draw3d_color(x: float, y: float, z: float, dt: int, image: ndarray, color: Tuple[float, float, float])

Draws a shape in 3d to a color image.

draw_mask(mask: Mask, x: float, y: float, z: float)

Draws a mask on the given drawing area.

ellipse() → Ellipse

Gets an ellipse describing the shape. Any 3D info is lost.

to_list() → List

Converts this shape to a list for serialization purposes. Convert back using from_list

volume() → float

Gets a 3-dimensional volume, in pixels^3
